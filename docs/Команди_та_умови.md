# Команди та умови мови блок-схем

## Команда PRINT V

**Синтаксис:** `PRINT V`

**Опис:** Вивести значення спільної змінної V на стандартний вивід. V має бути оголошена в секції `VARS:`.

**Приклад у потоку:**
```
2: PRINT x -> -1
```
Виконується крок 2: виводиться поточне значення змінної `x`, після чого потік переходить до кроку -1 (завершення).

---

## Умови у блоках розгалуження

У блоках розгалуження допускаються дві форми умов.

### 1. Порівняння V==C

**Синтаксис:** `V == C`

**Опис:** Умова «значення V рівне значенню C», де V — спільна змінна, C — цілочисельний літерал. Якщо умова істинна, виконання продовжується з вузла *trueNextId*; якщо хибна — з вузла *falseNextId*.

**Формат рядка в описі потоку:** після `->` вказуються **два** номери кроків через кому: спочатку крок для випадку «умова істинна», потім для «умова хибна».

**Приклад:**
```
4: x == 0 -> 5, 6
```
- Якщо `x` дорівнює 0, перехід на крок 5.
- Інакше перехід на крок 6.

---

### 2. Порівняння V<C

**Синтаксис:** `V < C`

**Опис:** Умова «значення V менше значення C», де V — спільна змінна, C — цілочисельний літерал. Якщо умова істинна, виконання продовжується з вузла *trueNextId*; якщо хибна — з вузла *falseNextId*.

**Формат рядка:** так само, після `->` два номери через кому: trueNextId, falseNextId.

**Приклад:**
```
2: x < 0 -> 3, 4
```
- Якщо `x` менше 0, перехід на крок 3.
- Інакше перехід на крок 4.

---

## Повний приклад програми

Файл `samples/correct_conditions.txt`:

```
VARS: x

THREAD T1:
  1: INPUT x -> 2
  2: x < 0 -> 3, 4
  3: PRINT x -> -1
  4: x == 0 -> 5, 6
  5: PRINT x -> -1
  6: PRINT x -> -1

TEST:
INPUT: 0
EXPECTED: 0
K: 10
```

**Пояснення виконання:**

1. Крок 1: зчитується значення в `x` з входу (наприклад, 0).
2. Крок 2: умова `x < 0` — хибна (0 не менше 0), тому перехід на крок 4.
3. Крок 4: умова `x == 0` — істинна, перехід на крок 5.
4. Крок 5: виконується `PRINT x` — на вивід потрапляє 0, потік завершується (перехід на -1).

Таким чином умови **V==C** та **V<C** коректно обробляються симулятором, а команда **PRINT V** виводить поточне значення спільної змінної на стандартний вивід.
